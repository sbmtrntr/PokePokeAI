## Usage
1. `statics/template_decks.json`に使いたいデッキを登録
2. `main.py`の`Player1_field`と`Player2_field`を変更
3. コマンドラインで以下を実行
```bash
python main.py
```

## Environment
対戦するだけなら（今のところ）必要なし
```bash
conda create -n pokepoke python=3.9
conda activate pokepoke
pip install -r requirements.txt
```
